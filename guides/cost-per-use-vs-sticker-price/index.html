Cost per use vs sticker price: buy the wear, not the tag

Sticker price answers “what do I pay today?” Cost per use answers “what do I pay for each time this thing does its job?”

Cost per use = (Price + lifetime upkeep) / Expected uses

Item typePreferWhy
ConsumablesUnit priceQuantity is the product
DurablesCost per useLifespan dominates
Hybrid (refills)BothDevice + consumable system

Worked example

Boots sketch. $60 boots × 40 wears = $1.50/wear. $220 boots × 200 wears = $1.10/wear — cheaper per use if the durability assumption is real.

Upkeep belongs in the numerator

Filters, sharpening, batteries, resoling — if you will pay it, include it. Excluding upkeep is how “cheap” printers become expensive.

Be honest about expected uses

Inflating lifespan to justify a splurge is the buyer version of fake seller margins. Use conservative wears based on your actual habits.

Checklist

  • Price entered
  • Upkeep estimated
  • Uses estimated conservatively
  • Cost/use compared across options
  • Return window noted if unsure

FAQ

Is cost per use the same as cost per day?

Related. Cost per day = total ownership cost / days kept. Use the denominator that matches how you consume the product.

What if I might not use it much?

Then expected uses fall and expensive durables look worse — correctly. Do not buy capacity you will not use.

Can unit price and cost per use disagree?

Yes. A cheaper-per-oz soap in a dispenser system might lose on cost-per-handwash after wasted product — rare, but possible. Pick the metric that matches the job.
