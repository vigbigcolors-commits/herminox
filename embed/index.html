3) Deep-link prefill (no widget needed)

Share a URL that opens the full calculator with numbers already filled. Strict uniqueness rule: Google should index only the clean tool URL (no query string). Prefill links like ?price=35&cogs=10 work for people, but are sent with noindex so they never become duplicate pages in search.

  • FBA example prefill → indexes as /sellers/fba-calculator/ only
  • ACoS example prefill → indexes as /sellers/acos-breakeven/ only

FBA params: price, cogs, weight, unit (oz|lb|g|kg), category, tier, fuel (0|1)

ACoS params: price, cogs, referral, fba, target, conv, organic

Unknown or out-of-range values are ignored. No script injection via query strings.

4) Partner rules (keep everyone safe)

  • Do not wrap the widget in pages that phish Seller Central logins.
  • Do not claim Herminox affiliation with Amazon.
  • Keep UTM tags honest (utm_source = your brand).
  • Widget estimates are educational — always tell readers to confirm fees in Seller Central.
